# Crowdfunding Application
This application allows users to create and contribute to crowdfunding campaigns with gasless transections.

## Tech Stack
 - Vite.js
 - Thirdweb
 - Ethers.js
 - Solidity
 - Hardhat
 - Openzeppelin
 - Biconomy
